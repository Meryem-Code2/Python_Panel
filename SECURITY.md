# Security Policy

## Supported Versions

Only the **latest version** of this project is supported with security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest (main)  | ✅ Yes    |
| Older versions | ❌ No     |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it responsibly by contacting the repository maintainer.

### How to Report
Please include:
- A clear description of the vulnerability
- Steps to reproduce the issue (if possible)
- Affected files or components
- (Optional) Suggestions for fixing or mitigating the issue

### Where to Report
- Via **GitHub private message**, or  
- Via **email** (if listed on the maintainer’s GitHub profile)

### Response Expectations
This is a **personal / educational project**, so response times may vary.  
Best effort will be made to:
- Acknowledge the report
- Evaluate the severity
- Fix the issue if necessary

### Disclaimer
This project is intended for **local and educational use only**.  
It is **not designed for production environments**, especially when handling:
- API keys
- Financial or banking data

Use at your own risk.
